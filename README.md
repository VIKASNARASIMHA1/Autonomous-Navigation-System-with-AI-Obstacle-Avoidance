# Autonomous Navigation System with AI Obstacle Avoidance
A comprehensive software-based autonomous navigation system that simulates embedded systems, computer vision, and reinforcement learning for intelligent obstacle avoidance and path planning.

## Table of Contents

## Overview

This project implements a complete autonomous navigation system that combines:

Embedded System Simulation - Virtual sensors and camera

Computer Vision - AI-powered object detection

Reinforcement Learning - DQN-based decision making

Control Systems - Path planning and obstacle avoidance
