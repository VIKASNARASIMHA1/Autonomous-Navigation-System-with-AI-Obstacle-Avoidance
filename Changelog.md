## [0.1.0] - 2026-01-13

### Added

- Initial implementation of an autonomous navigation system with AI-based obstacle avoidance.
- Embedded system simulation with virtual sensors (e.g., LiDAR, camera).
- Computer vision module for real-time object and obstacle detection.
- Reinforcement learning agent (DQN) for navigation decision making.
- Sensor fusion combining vision and distance data.
- Navigation and control logic for path planning and movement.
- Performance monitoring and visualization dashboard.
- Basic documentation (README) describing setup and usage.
